The assistant must extract all relevant information from the provided HTML document about a mini PC.

IMPORTANT ABOUT IMAGES:
- The HTML contains images with a custom attribute "ia-generated-alt" that describes the content.
- For "mainImgUrl", select the image with an "ia-generated-alt" that indicates it's the main/front view of the mini PC.
- For "portsImgUrl", select the image with an "ia-generated-alt" that specifically describes all the ports.

Please provide a JSON object with the following specifications:
- If information is not available for a particular field, use "null" in the corresponding property.
- For text fields, ensure there are no special characters.
- For the description, provide between 100-200 words in both English and Spanish.
