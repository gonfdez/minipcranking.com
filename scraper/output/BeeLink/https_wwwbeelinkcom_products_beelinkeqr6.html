EQR6 AMD Ryzen 6600H - A nice product

I've decided to buy an EQR6 AMD Ryzen 6600H from Beelink because in 2018 I've bought a BT3-PRO, an old mini PC from Beelink (that I've now replaced with it).

So I was looking for a mini PC but since the BT3-PRO had an Intel Atom (2-5 W of TDP, so it didn't need much power) and it was too slow for my needs (I use this mini PC as HTPC, I connect it to the TV, so I can see Netflix, Prime Video but also normal TV + as a home server), I've decided to make an upgrade choosing a Mini PC with an AMD Ryzen CPU.

This mini PC is very small, it's quiet but also very powerful, thanks to the CPU and the Crucial M.2 SSD. It has a lot of RAM, so you can use it for everything, from basic office work and multimedia to something like a mini home server, it could also run some VMs with Proxmox.

Since it's an AMD CPU, the integrated graphic is very good and it should be suitable for some games. You could make a retro-console with it, in the future I may consider doing this.

I use Linux Mint with it, it comes with Windows 11, but since I don't like Windows, I removed it.

I can confirm full support on GNU/Linux, at least on kernel 6.11.

Wi-FI & Bluetooth fully working out-of-the-box on Linux 6.11, so I was very happy to see it.

I'm very happy with this product, I hope it could last me for at least 5 years and more. The BT3-PRO that I've bought from Beelink is still working nicely and I've bought it in 2018, now we're in April 2025.

I appreciate Beelink products, the BIOS is fully customizable with a lot of options and if you want to use Windows, it is without bloatware from them.

Since I'm based in Italy, it took 4 days with DHL Express. DHL Express took care of customs clearance, so I've payed the various taxes via them.
