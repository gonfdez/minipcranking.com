# Mi Primer Post con Next.js y MDX

¡Bienvenido a mi blog! Este es un ejemplo de cómo se ve un post completo escrito en **MDX**. MDX combina la simplicidad de Markdown con el poder de React.

## ¿Qué es MDX?

MDX te permite escribir JSX directamente en tus archivos Markdown. Esto significa que puedes:

- Usar **texto en negrita** y _cursiva_
- Crear listas como esta
- Incluir bloques de código
- Y mucho más...

## Ejemplo de Código

Aquí tienes un ejemplo de código JavaScript:

```javascript
function saludar(nombre) {
  return `¡Hola, ${nombre}! Bienvenido a mi blog.`;
}

const mensaje = saludar("Desarrollador");
console.log(mensaje);
```

Y también código React:

```tsx
import { useState } from "react";

function Contador() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Has hecho clic {count} veces</p>
      <button onClick={() => setCount(count + 1)}>Incrementar</button>
    </div>
  );
}
```

## Lista de Tecnologías

Algunas de las tecnologías que estoy usando en este proyecto:

1. **Next.js 14** - Framework de React para producción
2. **MDX** - Markdown con superpoderes JSX
3. **Tailwind CSS** - Framework de CSS utilitario
4. **shadcn/ui** - Componentes de UI hermosos y accesibles
5. **TypeScript** - JavaScript con tipos estáticos
